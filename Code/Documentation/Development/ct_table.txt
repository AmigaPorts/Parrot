Table Asset
# Table Asset (ASSET_TABLE)

Design Version
  2.0
Code Version
  1.0

Asset Lookup Table

## C Version

  struct ASSET_TABLE
  {
    ULONG   at_AssetType;
    UWORD   at_Chapter;
    UWORD   at_Count;
    UWORD   at_Lowest;
    UWORD   at_Highest;
    +       ITEMS
  };

  struct ASSET_TABLE_ITEM
  {
    UWORD   ti_Id;
    UWORD   ti_Archive;
  };

## Fields

### at_Type

FourCC type of the Asset

example:
  'R', 'O', 'O', 'M'

### at_Chapter

The Chapter that these Assets should be used in or 0 for always loaded.

example:
  1

### at_Count

Number of Assets in this table

example:
  10

### at_Lowest

The Lowest Id number in this table

example:
  1

### at_Highest

example:
  10

### DATA

An array of ASSET_TABLE_ITEMS which are immediately after the ASSET_TABLE
in memory or disk. 

There should be exactly, at_Count of them. They do not need to be zero-
terminated.

example:
  { 1, 1 }
  { 2, 1 }
  { 3, 2 }
  { 4, 2 }
  { 5, 2 }
  { 6, 3 }
  { 7, 3 }
  { 8, 3 }
  { 9, 3 }
  { 10, 4 }
